Linha do tempo dos SO

  • 1956
    GM-NAA I/O
    O primeiro sistema operacional, chamado de GM-NAA I/O, foi desenvolvido para o computador IBM 704 pela IBM. Era um sistema operacional simples que gerenciava a entrada e saída de dados.
  • 1961
    CTSS
    O sistema operacional CTSS (Compatible Time-Sharing System) foi desenvolvido pelo MIT para o computador IBM 7094. Ele permitia que vários usuários usassem o computador ao mesmo tempo, uma ideia revolucionária na época.
  • 1964
    System/360 OS
    O sistema operacional IBM System/360 OS foi lançado para acompanhar o lançamento do computador IBM System/360. Era um sistema operacional completo, capaz de executar programas em diferentes linguagens de programação.
  • 1969
    UNIX
    Ken Thompson e Dennis Ritchie, da Bell Labs, desenvolveram o sistema operacional UNIX, originalmente escrito em linguagem assembly para o computador DEC PDP-7. UNIX se tornou um dos sistemas operacionais mais influentes da história.
  • 1975
    MS-DOS
    Bill Gates e Paul Allen fundaram a Microsoft e desenvolveram uma versão da linguagem BASIC para o computador Altair. Isso levou ao desenvolvimento do sistema operacional MS-DOS, lançado em 1981.
  • 1984
    Mac OS
    A Apple lançou o sistema operacional Mac OS, criado para o computador Macintosh. Ele apresentava uma interface gráfica de usuário, tornando-o mais fácil de usar do que outros sistemas operacionais baseados em texto.
  • 1991
    Linux
    Linus Torvalds lançou a primeira versão do Linux, um sistema operacional gratuito e de código aberto. Linux se tornou popular entre os usuários de computadores que procuravam uma alternativa ao Windows e ao Mac OS.
  • 1995
    Windows 95
    A Microsoft lançou o Windows 95, que se tornou um enorme sucesso comercial. Ele apresentava uma interface gráfica mais aprimorada, suporte para multitarefa e um sistema de arquivos mais avançado.
  • 2001
    Mac OS X
    A Apple lançou o Mac OS X, uma nova versão do sistema operacional Mac que era baseada no kernel UNIX. Isso tornou o Mac OS X mais estável e seguro do que as versões anteriores.
  • 2009
    Chrome OS
    O Google lançou o Chrome OS, um sistema operacional baseado em nuvem que foi projetado para ser usado em computadores portáteis e dispositivos móveis.
  • 2015
    Windows 10
    A Microsoft lançou o Windows 10, que foi projetado para ser usado em uma ampla variedade de dispositivos, incluindo computadores, tablets e smartphones. Ele apresentava uma nova interface de usuário, suporte para várias áreas de trabalho e um assistente virtual chamado Cortana.
  • 2021
    macOS Big Sur
    O sistema operacional macOS Big Sur foi lançado pela Apple, com um design completamente novo e uma interface de usuário atualizada. Ele apresenta novos recursos de privacidade e segurança, bem como melhorias de desempenho.
